# 🚀 React ES6 CRUD App

A modern **Task Manager (CRUD)** application built using **React Hooks** and **ES6+ JavaScript features**.
This project demonstrates clean code practices, modern JavaScript syntax, and a simple yet elegant UI.

---

## ✨ Features

* ➕ Add tasks
* 📋 View tasks
* ✏️ Edit tasks
* ❌ Delete tasks
* 💾 Persistent storage using LocalStorage
* 🎨 Modern dark UI design
* ⚡ Smooth user experience with React Hooks

---

## 🧠 JavaScript vs ES6 (Quick Comparison)

| 🔙 Old JavaScript                   | 🚀 ES6+ (Modern JavaScript)          |
| ----------------------------------- | ------------------------------------ |
| `var`                               | `let`, `const`                       |
| `function()`                        | Arrow functions `() => {}`           |
| String concatenation `"Hi " + name` | Template literals `` `Hi ${name}` `` |
| No destructuring                    | `{ name } = obj`                     |
| Manual loops (`for`)                | `.map()`, `.filter()`                |
| No modules                          | `import/export`                      |
| `this` confusing behavior           | Lexical `this` (cleaner)             |
| Object mutation                     | Spread operator `{ ...obj }`         |

👉 This project mainly uses **ES6+ features**, which are standard in modern React development.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/MiyuruWijesinghe/react-es6-crud.git
cd react-es6-crud
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
